# QZ Tray Setup - USB Chek Apparatdan Chek Chiqarish

## Qanday ishlaydi?

1. **Backend** (Server/Hosting) - faqat ma'lumot yuboradi
2. **Frontend** (Vercel/Browser) - ma'lumotni qabul qiladi
3. **QZ Tray** (Client kompyuter) - USB printerga yuboradi

Bu hostingda ham ishlaydi, chunki printer **client-side** (kompyuterda).

---

## O'rnatish

### 1. QZ Tray Desktop App o'rnatish

Har bir kompyuterda (chek chiqaradigan joyda) QZ Tray o'rnatilishi kerak:

1. **Yuklab olish:**
   - https://qz.io/download/ ga kiring
   - Windows/Linux/Mac uchun versiyani yuklab oling

2. **O'rnatish:**
   - O'rnatish faylini ishga tushiring
   - Default sozlamalar bilan o'rnating

3. **Ishga tushirish:**
   - QZ Tray tray icon (taskbar) da ko'rinishi kerak
   - Agar ko'rinmasa, Start menudan qidiring

### 2. Printer sozlash

1. USB printer ni kompyuterga ulang
2. Windows/Linux/Mac da printer driver o'rnatilgan bo'lishi kerak
3. QZ Tray da printer tanlash:
   - QZ Tray icon ga bosing
   - "Printer" > "Set Default Printer"
   - Yoki kodda printer nomini belgilash mumkin

---

## Ishlatish

### Admin panelda:

1. **Yakunlangan order** ni oching
2. **"Chek chiqarish"** tugmasini bosing
3. QZ Tray avtomatik printerga yuboradi

### Birinchi marta:

- Browser QZ Tray ga ruxsat so'raydi
- "Allow" tugmasini bosing
- Keyingi safar avtomatik ishlaydi

---

## Xatoliklar

### "QZ Tray o'rnatilmagan" xatosi:

- QZ Tray desktop app o'rnatilganligini tekshiring
- QZ Tray ishga tushirilganligini tekshiring (tray icon)
- Browser ni qayta yuklang

### "Printer topilmadi" xatosi:

- USB printer ulanganligini tekshiring
- Printer driver o'rnatilganligini tekshiring
- QZ Tray da default printer tanlanganligini tekshiring

### "Connection failed" xatosi:

- QZ Tray WebSocket port (8181) ochiqligini tekshiring
- Firewall QZ Tray ga ruxsat berganligini tekshiring

---

## Production uchun

✅ **Hostingda ishlaydi:**
- Backend serverda (eskiz.uz)
- Frontend Vercelda
- QZ Tray client kompyuterda

✅ **Real-time:**
- Order yakunlanganda darhol chek chiqarish mumkin
- Har bir kompyuter o'z printeriga chiqaradi

✅ **Xavfsiz:**
- QZ Tray faqat local printerga ulashadi
- Server printerga to'g'ridan-to'g'ri ulashmaydi

---

## Qo'shimcha ma'lumot

- QZ Tray Documentation: https://qz.io/docs/
- ESC/POS Format: https://en.wikipedia.org/wiki/ESC/POS
- Support: https://qz.io/support/
